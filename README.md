# HATchat 🎩💬

Реал-тайм чат-приложение, работающее на протоколе WebSockets. Проект позволяет пользователям мгновенно подключаться к различным комнатам, общаться без задержек, видеть список активных участников и отслеживать, когда кто-то набирает текст.

A real-time chat application powered by WebSockets. The project allows users to instantly join chat rooms, communicate without delays, view online participants, and see live typing indicators.

---

## 🚀 Функционал / Features

* **Real-time обмен сообщениями:** Мгновенная доставка сообщений без перезагрузки страницы (Socket.io).
* **Комнаты (Rooms):** Возможность создавать изолированные комнаты для общения.
* **История чата:** Сохранение последних 50 сообщений при входе в комнату.
* **Индикатор "Печатает...":** Живое отображение статуса ввода текста другими пользователями.
* **Онлайн-пользователи:** Отображение списка участников, находящихся в комнате прямо сейчас.
* **Адаптивный интерфейс:** Чат корректно отображается как на ПК, так и на смартфонах.

---

## 🛠 Стек технологий / Tech Stack

* **Backend:** Node.js, Express, Socket.io
* **Frontend:** Vanilla HTML5, CSS3, JavaScript (Браузерный Socket.io клиент)
